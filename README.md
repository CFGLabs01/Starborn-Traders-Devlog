# 🌌 Starborn Traders – Devlog Repository

Welcome to the official devlog repository for **Starborn Traders**, a text-based space exploration and trading game developed by Carlos Felipe Garcia (CFG Labs).

This repo serves as a **transparent and chronological development journal**, documenting daily milestones, challenges, reflections, and implementation notes throughout the project lifecycle.


## 🔍 Purpose

- ✅ Maintain development accountability  
- 🧠 Encourage structured thinking and iteration  
- 🚀 Demonstrate capability in full-stack game logic and testing  
- 📜 Provide an open-source track record (MIT license)  
- 💡 Inspire others navigating solo dev journeys with limited resources
